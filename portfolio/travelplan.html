Travel Plan

A basic React app integrated with Google's Firebase



Continuing my React journey, I created a basic React app with a Firebase backend. This app functions as a basic project creator , where users can log in via Firebase's auth function and create a project consisting of title and some content.

The focus of this project was integrating the front end with Firebase. In addition to authentication, it uses Firebase security to prevent modification of data by users who are not signed in, and also persists created projects in Google's Cloud Firestore.

It uses Firebase Cloud Functions to create projects and activity notifications. The activity panel on the homepage updates in real time as users sign up and create projects, showcasing an example of realtime cloud functions.
