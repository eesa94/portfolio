Seoul Food

React application for a Korean restaurant



Built on React, this project is an online presence for a fictional restaurant specialising in Korean cuisine.

I have thoroughly enjoyed learning React over the last few weeks, and I was keen to put my newly-learned skills into practice.

Content-wise, I kept it simple, because the focus was on learning how React functions. This app will act as a blueprint from which I will continue to learn and develop my knowledge in this area.
