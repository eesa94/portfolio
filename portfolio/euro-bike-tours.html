Euro Bike Tours

Euro Bike Tours

A fully-functional e-commerce store for a cycle travel company



This is the main project which was completed as part of my bootcamp. It is an e-commerce store built on Rails, featuring user authentication, authorisation, comments, caching, testing, security, and asynchronous request handling, amongst others.

The idea for this project came from exploring travel providers in my spare time. I decided to create an app for a fictional travel agent which provides cycling tours around Europe.

Integrating the front end responsiveness of Bootstrap with the back end of Rails allowed me to create a simplified version of what would be a fully-functioning online store, where users could browse different tour packages and pay using Stripe.

The primary focus of this project was the back end; the goal being to build a robust app which would meet all of the functional requirements on the back end, upon which the front end could be built.
